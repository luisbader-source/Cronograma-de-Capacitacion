# Cronograma de Capacitación - Exploración de Diseño

## Contexto
Aplicación web para visualizar el cronograma de capacitación de estudiantes de la Universidad de Córdoba, organizado por facultades. Acceso vía Zoom, de lunes a viernes, 9-10 AM.

### Idea 1: Modernismo Corporativo Elevado
**Design Movement**: Bauhaus contemporáneo con influencia de diseño corporativo nórdico

**Core Principles**:
- Claridad radical: Cada elemento tiene propósito, sin decoración innecesaria
- Jerarquía tipográfica fuerte: Títulos en sans-serif bold, cuerpo en weight 400
- Espaciado generoso: Respiración visual entre secciones
- Monocromía con acentos: Blanco/gris neutro + verde institucional de Unicórdoba

**Color Philosophy**:
- Fondo: Blanco puro (#FFFFFF) con toques de gris muy claro (#F8F9FA)
- Primario: Verde Unicórdoba (#1B7D3D) para CTAs y énfasis
- Secundario: Gris oscuro (#2C3E50) para textos
- Acentos: Tonos de verde más claros para estados hover/active
- Intención emocional: Profesionalismo, confianza, institucionalidad

**Layout Paradigm**:
- Grid asimétrico de 2 columnas: Panel lateral con facultades (25% ancho) + contenido principal (75%)
- Tarjetas modulares para cada día con bordes sutiles
- Separadores visuales minimalistas (líneas finas, no bloques)

**Signature Elements**:
1. Icono de calendario minimalista en header
2. Línea vertical de progreso semanal (lunes a viernes)
3. Badges de facultad con tipografía uppercase y peso bold

**Interaction Philosophy**:
- Transiciones suaves (200ms) en hover
- Cambio de color en facultades seleccionadas
- Efecto de "card lift" sutil en tarjetas de horario

**Animation**:
- Entrada de tarjetas con fade-in + slide-up (300ms, ease-out)
- Línea de progreso que se anima al cargar
- Hover en facultades: cambio de color + sombra sutil

**Typography System**:
- Display: Poppins Bold (36px) para títulos principales
- Heading: Poppins SemiBold (24px) para secciones
- Body: Inter Regular (16px) para contenido
- Caption: Inter Regular (12px) para metadatos
- Jerarquía: Bold > SemiBold > Regular en weights

### Idea 2: Diseño Educativo Cálido y Accesible
**Design Movement**: Pedagogía visual moderna con influencia de diseño educativo contemporáneo

**Core Principles**:
- Inclusividad visual: Colores vibrantes pero accesibles (WCAG AA+)
- Narrativa visual: Cada sección cuenta una historia del proceso de capacitación
- Interactividad lúdica: Elementos que invitan a explorar sin ser infantiles
- Tipografía legible: Prioridad en readability para audiencias diversas

**Color Philosophy**:
- Fondo: Blanco cálido (#FAFBF8) con gradiente muy sutil
- Primario: Azul educativo (#0066CC) para acciones
- Secundario: Naranja cálido (#FF8C42) para destacados
- Terciario: Verde suave (#4CAF50) para confirmación
- Intención: Calidez, apertura, accesibilidad, energía positiva

**Layout Paradigm**:
- Diseño fluido con secciones full-width separadas por espacios
- Tarjetas grandes (casi hero) para cada facultad
- Timeline visual vertical que muestra progresión de lunes a viernes
- Uso de iconografía ilustrada para cada facultad

**Signature Elements**:
1. Ilustraciones custom de estudiantes por facultad
2. Timeline vertical con nodos interactivos
3. Tarjetas con fondo degradado suave por facultad

**Interaction Philosophy**:
- Feedback inmediato y visual en cada interacción
- Animaciones que comunican (no solo decoran)
- Tooltips informativos en hover

**Animation**:
- Timeline que se dibuja al cargar (stroke animation)
- Tarjetas que aparecen con bounce suave
- Iconos que animan al hover (rotate, scale)

**Typography System**:
- Display: Montserrat Bold (40px) para títulos
- Heading: Montserrat SemiBold (28px) para facultades
- Body: Open Sans Regular (16px) para contenido
- Caption: Open Sans Regular (13px) para detalles
- Jerarquía clara con variación de color además de peso

### Idea 3: Minimalismo Elegante con Énfasis en Contenido
**Design Movement**: Swiss Design + Contemporary Minimalism

**Core Principles**:
- Contenido es rey: Diseño se retira para dejar brillar la información
- Precisión geométrica: Grid estricto, alineación perfecta
- Tipografía como ornamento: Variación de tamaño y peso crea ritmo visual
- Sutileza en detalles: Bordes finos, espacios precisos, colores desaturados

**Color Philosophy**:
- Fondo: Crema muy clara (#FFFBF7)
- Primario: Negro suave (#1A1A1A) para textos
- Secundario: Beige cálido (#D4C5B9) para separadores
- Acentos: Rojo oscuro (#8B3A3A) para CTAs y énfasis
- Intención: Elegancia, sofisticación, enfoque en contenido

**Layout Paradigm**:
- Grid de 12 columnas con márgenes amplios
- Tarjetas sin bordes, solo separadas por espaciado
- Tipografía grande y respirable
- Simetría deliberada con asimetrías estratégicas

**Signature Elements**:
1. Líneas divisoras de grosor variable
2. Números grandes de día en tipografía serif
3. Badges minimalistas sin relleno (solo borde)

**Interaction Philosophy**:
- Cambios sutiles en opacidad y espaciado
- Transiciones muy lentas (400ms) para sensación de lujo
- Hover revela información adicional

**Animation**:
- Fade-in lento de elementos (500ms)
- Cambio de opacidad en hover (no color)
- Líneas que se animan al cargar

**Typography System**:
- Display: Playfair Display Bold (48px) para títulos
- Heading: Playfair Display SemiBold (32px) para secciones
- Body: Lato Regular (17px) para contenido
- Caption: Lato Regular (13px) para metadatos
- Jerarquía: Serif para títulos, sans-serif para cuerpo (contraste elegante)

## Decisión Final

**Seleccionado: Idea 1 - Modernismo Corporativo Elevado**

Este enfoque es el más apropiado para una institución educativa oficial como la Universidad de Córdoba porque:

1. **Profesionalismo**: Transmite confianza y seriedad, esencial para un cronograma oficial
2. **Claridad**: La jerarquía visual fuerte facilita que los estudiantes encuentren rápidamente su facultad y horario
3. **Institucionalidad**: El verde de Unicórdoba y el diseño nórdico refuerzan la identidad de la universidad
4. **Accesibilidad**: El contraste alto y la tipografía clara garantizan legibilidad para todos
5. **Escalabilidad**: El sistema es flexible para agregar más facultades o información sin perder coherencia

**Paleta de colores final**:
- Verde Unicórdoba: #1B7D3D (primario)
- Blanco: #FFFFFF (fondo)
- Gris claro: #F8F9FA (fondos secundarios)
- Gris oscuro: #2C3E50 (textos)
- Verde claro: #4CAF50 (estados hover)

**Tipografía**:
- Poppins (títulos y énfasis)
- Inter (cuerpo y detalles)
